About Swiftplot

A Tool for (Data) Scientists

If you’re a scientist or you work with data on a regular basis, then you might have encountered this problem: you have a list of numbers that you want to plot, but you can’t do it immediately. Maybe you can’t afford another license for expensive software packages: MATLAB, Igor, Origin, or Excel. Or maybe you have the software, but Excel’s plots are too ugly, and you can’t remember the MATLAB code. Either way, plotting data is such a simple task that you shouldn’t need to install anything to your computer. We wanted to develop a better way.

With Swiftplot, you can simply go to swiftplot.com and drag your data file to the browser. Instantly, you’ll get a high quality plot that you can customize for your presentations.

Democratizing Data Analysis

Numbers don’t lie. Yet we aren’t always able to look at the raw data when we’re reading about problems in the newspaper. Don’t believe that global temperatures are rising? You should be able to click a link and see the data for yourself, and create a trend line that extrapolates to the year 2100, right there in the browser.

Open Source

There any number of things you might want to do with your data. Many tasks, such as fitting trend lines or Gaussians, are extremely common. Others are rare. Either way, if someone else has written the code, you shouldn’t have to. That’s why this project is fully open source, with a growing list of analysis tools to which anyone can contribute.
